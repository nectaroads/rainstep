# About
Rainstep is a Multiplayer Bullet-Hell game. Gather your friends to survive the most you can.
The project is majorly a RPC, Sockets and Microservices study, due academic reasons. 

## Features
- Choose your name
- Connect with a Friend

## Technologies
- HTML
- CSS
- Javascript
